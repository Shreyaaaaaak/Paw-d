# Paw'd 🐾

A playful pet matching web app that lets users swipe through dog and cat profiles with profile customization, boost mode, streak tracking, distance labels, statistics, and background audio.

## Features

- Swipeable pet cards with like, nope, and super like
- Profile page for avatar and display name
- Boost mode to surface premium pet profiles first
- Fake distance text on each pet card
- Swipe streak counter and stats dashboard
- Background ambient music and swipe sound effects
- Match list with chat history

## Files

- `index (1).html` — main UI layout
- `style.css` — app styling
- `script.js` — app logic and state
- `pets.js` — pet profile data

## Usage

1. Open `index (1).html` in a browser.
2. Use the buttons to start swiping, open profile, or view stats.
3. Toggle music and boost features from the home screen.

## Notes

- The app stores matches, profile settings, and stats in localStorage.
- No build process is required; this is a static web app.
